# 110 Cinemas

A simple web application for browsing movies and reserving cinema seats

## Tech stack
- **Backend**: Python / Django + HTMX
- **Database**: SQLite
## Setup
```bash
uv venv
source .venv/bin/activate
uv sync
```

## Run Linter
```bash
uv run pylint cinema
```

## Run Tests
```bash
uv run pytest --cov
```
